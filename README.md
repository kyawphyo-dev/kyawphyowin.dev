# 🌐 Kyaw Phyo Win — Portfolio Website

## 📌 Overview

This is my personal portfolio website built to showcase my skills, projects, and experience as a **Full-Stack Developer**.

The goal of this project is to:

- Present my technical skills clearly
- Highlight my best and latest projects
- Provide an easy way for recruiters to contact me

---

## 🚀 Tech Stack

### Frontend

- React.js
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)

### Backend (for future/full-stack integration)

- Laravel (API)

### Tools & Deployment

- Git & GitHub
- Vercel (deployment)

---

## ✨ Features

- 📱 Responsive design (mobile, tablet, desktop)
- 🌙 Dark / Light mode (Theme Context)
- 🧭 Scroll spy navigation
- 📌 Sticky navbar with hide/show on scroll
- 🎬 Smooth animations (Framer Motion)
- 📂 Reusable components
- 📬 Contact section with social links

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/kyawphyo-dev/kyawphyowin.dev.git
```

2. Navigate to the project folder

```bash
cd kyawphyowin.dev
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

---

## 🌍 Deployment

This project is deployed using **Vercel**.

Steps:

1. Push code to GitHub
2. Connect repository to Vercel
3. Deploy automatically
